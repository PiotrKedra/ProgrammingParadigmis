## hs-skeleton-project

```bash
  - cabal update
  - cabal sandbox init
  - cabal install --only-dpendencies --enable-tests
  - cabal configure --enable-tests
  

  - cabal test
  - cabal build
  - cabal run
  - .cabal-sandbox/bin/MyApp
  
```
